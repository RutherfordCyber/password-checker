# Password Checker 🔐

A simple Python tool that evaluates password strength based on basic security rules.

##Installation

Clone the repository and run the program:

```bash
git clone https://github.com/RutherfordCyber/password-checker.git
cd password-checker
python3 main.py
```

## Features
- Checks password length
- Detects weak vs strong passwords
- Encourages better security practices

## How to Run

```bash
python3 main.py
```

